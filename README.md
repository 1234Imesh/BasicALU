# 4-bit ALU using Ripple Carry Adder and 2:1 Multiplexers in SystemVerilog

A basic 4-bit Arithmetic Logic Unit (ALU) implemented in **SystemVerilog** using a **ripple carry adder** and **five 2:1 multiplexers**. The design demonstrates fundamental digital system concepts including modular arithmetic circuit design, carry propagation, multi-bit bus selection, and functional verification using a testbench.

## Overview

This ALU supports two operations:

- **Addition of two 4-bit inputs**
- **Zero output**

A single select signal controls the output:

- **SEL = 0** → output is `0000` and carry-out is `0`
- **SEL = 1** → output is `A + B` and carry-out is the adder carry

The arithmetic operation is performed using a **4-bit ripple carry adder**. The final ALU result is selected using **five individual 2:1 multiplexers**:

- **4 multiplexers** for the 4 output bits
- **1 multiplexer** for the carry-out

## Design Architecture

The design is organized hierarchically into the following modules:

1. **Full Adder**
   - Performs 1-bit addition with carry-in and carry-out

2. **4-bit Ripple Carry Adder**
   - Built by cascading four full adders
   - Produces a 4-bit sum and a final carry-out

3. **ALU Output Selection**
   - Uses five 2:1 multiplexers
   - Selects between zero and the adder result

## Functional Description

Let the two 4-bit inputs be:

\[
A = A_3A_2A_1A_0,\qquad B = B_3B_2B_1B_0
\]

The ripple carry adder computes:

\[
S = A + B
\]

The ALU output is defined as:

$$
Y =
\begin{cases}
0000, & \text{if } SEL = 0 \\
S, & \text{if } SEL = 1
\end{cases}
$$

The carry output is defined as:

$$
C_{out} =
\begin{cases}
0, & \text{if } SEL = 0 \\
C_{adder}, & \text{if } SEL = 1
\end{cases}
$$

where:

- **Y** is the 4-bit ALU output
- **S** is the 4-bit sum from the ripple carry adder
- **Cadder** is the carry-out generated by the ripple carry adder
